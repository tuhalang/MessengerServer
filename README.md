# MessengerServer
## Using
- IDE: eclipse
- JDK11

## Configuration
1. Open file **application.properties** in folder config
2. Change NUM\_OF\_THREAD and SERVER\_PORT

## Run
- Clone project

	git clone https://github.com/tuhalang/MessengerServer.git

- Open project by eclipse
- Run file HomeController
